# Restaurant Page 

This project part of [The Odin Project's](https://www.theodinproject.com) "Intermediate HTML and CSS" course .

[Project Requirements](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page)


## Technologies Used
 - HTML5
 - CSS3
 - JavaScript (ES6 Modules)
 - npm (Node package Manager)

## Features
- Dynamic tab switching between Home and Contact pages.
- Entirely rendered via JavaScript DOM manipulation.
- Responsive design for different screen sizes.
